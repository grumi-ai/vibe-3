공공직군 행정업무 슈퍼앱

팀 일정, 엑셀 자동화, 민원 대응, 뉴스 수집을 하나로 묶는 행정업무 지원 플랫폼의 요구사항, 아키텍처, 운영 기준을 한눈에 확인하는 종합 문서입니다.

팀원 스케쥴 관리

휴가, 근무, 출장, 교육 등 팀 일정을 캘린더에서 공유하고 팀원별, 유형별로 필터링합니다.

엑셀 업무 자동화

특정 컬럼 기준으로 엑셀을 분할하거나 여러 파일을 병합해 반복 작업을 줄입니다.

민원 대응 챗봇

첨부된 민원 매뉴얼을 바탕으로 대응 요약, 답변 초안, 검토 필요 사항을 제공합니다.

뉴스 기사 수집

공공 행정 관련 뉴스를 매일 아침 수집하고, 날짜와 키워드 기준으로 조회합니다.

MVP 범위

  • 팀 일정 CRUD 및 캘린더 조회
  • 엑셀 파일 분할 및 병합
  • 민원 매뉴얼 기반 답변 초안 생성 구조
  • 키워드 기반 뉴스 수집 및 목록 조회
  • SQLite 기반 로컬 데이터 저장

기술 구성

Frontend TypeScript, Vite, React, npm
Backend Python, FastAPI, Uvicorn, uv
Database SQLite

권장 구조

  • frontend/src/features/schedule: 일정 관리 UI
  • frontend/src/features/excel-automation: 엑셀 자동화 UI
  • frontend/src/features/complaint-chatbot: 민원 챗봇 UI
  • frontend/src/features/news: 뉴스 조회 UI
  • backend/app/api/routes: FastAPI 라우터
  • backend/app/services: 업무 로직
  • backend/app/repositories: SQLite 접근 계층

운영 주의사항

  • PowerShell에서는 npm 대신 npm.cmd 사용을 권장합니다.
  • uv 캐시 권한 문제가 있으면 프로젝트 내부 .uv-cache를 지정합니다.
  • 민원 챗봇 답변은 공식 답변이 아닌 담당자 검토용 초안으로 표시해야 합니다.
  • 업로드 파일은 처리 완료 후 삭제하는 정책을 기본으로 합니다.

주요 API 초안

영역 엔드포인트 역할
Schedule GET /api/schedules, POST /api/schedules 일정 조회 및 생성
Excel POST /api/excel/split, POST /api/excel/merge 엑셀 분할 및 병합
Complaint POST /api/complaints/manuals, POST /api/complaints/chat 매뉴얼 업로드 및 민원 답변 초안 생성
News GET /api/news, POST /api/news/collect 뉴스 목록 조회 및 수동 수집
