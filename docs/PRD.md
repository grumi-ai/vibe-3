## 1. 개요

공공직군 행정업무 슈퍼앱은 팀원 일정 관리, 엑셀 업무 자동화, 민원 대응 챗봇, 뉴스 기사 수집 기능을 한 곳에서 제공하는 내부 업무 지원 도구다.  
업무 중복을 줄이고, 반복적인 행정 작업을 빠르게 처리하며, 공공 행정 관련 정보를 매일 확인할 수 있도록 돕는 것이 목적이다.

## 2. 제품 목표

- 팀원들의 휴가, 근무, 출장 일정을 공유하고 조회할 수 있다.
- 엑셀 파일을 특정 컬럼 기준으로 분리하거나 합칠 수 있다.
- 민원 매뉴얼을 바탕으로 대응 스크립트와 안내문 초안을 만들 수 있다.
- 공공 행정 관련 뉴스를 매일 아침 자동으로 수집하고 확인할 수 있다.

## 3. 사용자

- 행정 담당자
- 민원 응대 담당자
- 일정 관리 담당자
- 자료 정리 및 보고 담당자
- 관리자 및 팀장

## 4. 핵심 기능

### 4.1 팀원 스케줄 관리

#### 설명
팀원들의 휴가, 근무, 출장 등 일정을 공유하는 캘린더 기능이다.

#### 주요 기능

- 팀원 등록, 수정, 삭제
- 일정 등록, 수정, 삭제
- 주간 뷰 제공
  - 리스트 또는 표 형식
  - 날짜별 일정 확인
- 월간 뷰 제공
  - 캘린더 형식
  - 월 단위 일정 요약 확인
- 팀원 필터
- 날짜 범위 필터
- 일정 상세 확인

#### 성공 기준

- 사용자는 팀원과 일정을 직접 관리할 수 있다.
- 사용자는 주간 뷰에서 이번 주 일정을 쉽게 확인할 수 있다.
- 사용자는 월간 뷰에서 한 달 일정을 한눈에 볼 수 있다.

### 4.2 엑셀 업무 자동화

#### 설명
특정 컬럼을 기준으로 엑셀 파일을 나누거나 합치는 기능이다.

#### 주요 기능

- `.xlsx`, `.xls`, `.csv` 업로드
- 기준 컬럼 선택
- 파일 분리
- 파일 병합
- 결과 다운로드
- 처리 실패 시 오류 안내

#### 성공 기준

- 사용자는 별도 편집 없이 엑셀 분리/병합 작업을 완료할 수 있다.
- 처리 결과 파일을 다운로드할 수 있다.

### 4.3 민원 대응 챗봇

#### 설명
첨부된 민원 매뉴얼을 바탕으로 대응 방식이나 스크립트를 작성해주는 챗봇이다.

#### 주요 기능

- PDF, DOCX, TXT 형식 매뉴얼 업로드
- 매뉴얼 내용 추출
- 민원 질문 입력
- 대응 초안 생성
- 검토가 필요한 부분 표시
- 검색형 질의와 생성형 답변의 분리

#### 성공 기준

- 사용자는 매뉴얼을 업로드하고 질문할 수 있다.
- 사용자는 답변 초안을 빠르게 확인할 수 있다.

### 4.4 뉴스 기사 수집

#### 설명
공공 행정 관련 뉴스를 아침마다 수집하는 기능이다.

#### 주요 기능

- 지정 키워드 기반 기사 수집
- 정해진 시간에 자동 수집
- 기사 제목, 출처, 요약, 링크 저장
- 최신 기사 목록 확인
- 기사 상세 또는 원문 이동

#### 성공 기준

- 사용자는 매일 수집된 뉴스를 확인할 수 있다.
- 중복 기사와 오류 수집을 최소화한다.

## 5. 범위

### 포함 범위

- 팀원 일정 관리
- 엑셀 자동화
- 민원 챗봇
- 뉴스 수집
- 공통 UI 레이아웃
- FE-BE 연동
- BE-DB 연동

### 제외 범위

- 외부 공공기관 계정 연동
- 모바일 앱 별도 개발
- 복잡한 권한 체계
- 장기 이력 분석 리포트

## 6. 우선순위

1. 팀원 스케줄 관리
2. 엑셀 업무 자동화
3. 민원 대응 챗봇
4. 뉴스 기사 수집
5. 공통 운영 기능

## 7. 완료 기준

- 네 가지 핵심 기능이 각각 독립적으로 동작한다.
- FE와 BE가 연결되어 있다.
- BE와 DB가 연결되어 있다.
- 기본적인 오류 처리와 빈 상태 처리가 제공된다.
